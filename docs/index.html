PRIVACIDADE

Política de Privacidade

CalculaAI — versão 1.0
Última atualização: maio de 2026

O CalculaAI não coleta, armazena nem compartilha dados pessoais de seus usuários. Esta política descreve em detalhes como o aplicativo trata as informações.

1. Sobre o aplicativo

O CalculaAI é um aplicativo de consulta de índices financeiros brasileiros (IPCA, IGP-M, SELIC e CDI) e simulação de rendimento de títulos de renda fixa (CDB). Os dados exibidos são obtidos exclusivamente de fontes públicas oficiais, como o Banco Central do Brasil (BCB).

2. Dados coletados

O CalculaAI não coleta dados pessoais identificáveis. Especificamente:

  • Não solicitamos nome, e-mail, CPF, telefone ou qualquer dado de identificação pessoal.
  • Não criamos contas de usuário.
  • Não rastreamos localização geográfica.
  • Não utilizamos ferramentas de analytics ou rastreamento de comportamento.
  • Não integramos serviços de publicidade.

3. Armazenamento local

O aplicativo pode salvar preferências do usuário (como idioma selecionado) diretamente no dispositivo, usando o armazenamento local do Android. Esses dados:

  • Ficam exclusivamente no dispositivo do usuário.
  • Não são transmitidos a nenhum servidor.
  • Podem ser removidos a qualquer momento ao desinstalar o aplicativo.

4. Idioma do dispositivo

O aplicativo lê a configuração de idioma do dispositivo (por exemplo, Português ou Inglês) para exibir o conteúdo no idioma preferido. Essa leitura é feita localmente e não é transmitida a servidores externos.

5. Fontes de dados financeiros

Os índices financeiros exibidos no CalculaAI são obtidos da API pública do Banco Central do Brasil (SGS — Sistema Gerenciador de Séries Temporais), acessível em api.bcb.gov.br. Nenhuma informação do usuário é enviada nessas consultas.

6. Serviços de terceiros

O CalculaAI utiliza infraestrutura da Cloudflare Workers para cache e distribuição das requisições à API do Banco Central. Nenhum dado pessoal do usuário é transmitido a esse serviço. Para mais informações, consulte a política de privacidade da Cloudflare.

7. Segurança

Como o aplicativo não coleta nem armazena dados pessoais em servidores, o risco de exposição de dados é mínimo. Todas as comunicações externas do aplicativo usam conexões seguras (HTTPS/TLS).

8. Crianças

O CalculaAI não é direcionado a crianças menores de 13 anos e não coleta informações de crianças.

9. Alterações nesta política

Eventuais alterações nesta política serão publicadas nesta página com atualização da data de vigência. O uso continuado do aplicativo após qualquer alteração implica aceitação da nova versão.

10. Contato

Para dúvidas sobre esta política de privacidade, entre em contato pelo GitHub: github.com/matheus-vmattos.

CalculaAI é um projeto independente desenvolvido por Matheus Mattos.
Os dados financeiros são fornecidos pelo Banco Central do Brasil e têm caráter meramente informativo.
