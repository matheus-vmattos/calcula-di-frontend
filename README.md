# CalculaAI

CalculaAI é um aplicativo mobile focado em consulta e cálculo de indicadores financeiros brasileiros, com uma experiência simples, rápida e visualmente limpa.

O projeto nasceu para centralizar informações úteis do mercado, como índices do Banco Central do Brasil, simulações com CDI e cálculos de marcação a mercado, em uma interface direta para quem quer tomar decisões com mais clareza.

## Visão Geral

O CalculaAI entrega uma experiência mobile para acompanhar indicadores e realizar cálculos financeiros sem depender de planilhas complexas ou consultas manuais.

A proposta do app é combinar dados financeiros relevantes, interface objetiva e cálculos rápidos em uma experiência mobile pronta para uso e publicação.

## Funcionalidades

- Consulta de índices financeiros
- Visualização de indicadores como CDI, SELIC e IPCA
- Calculadora baseada em CDI
- Tela de cálculo para Prefixado/MTM
- Alternância entre português e inglês
- Identidade visual própria da marca CalculaAI
- Splash/entrada visual com marca do app

## Identidade Visual

A identidade visual do CalculaAI utiliza uma estética minimalista e tecnológica, com fundo preto, marca em branco e acento ciano.

A proposta visual segue uma linha premium, objetiva e silenciosa: transmitir confiança, clareza e sofisticação sem excesso de elementos.
